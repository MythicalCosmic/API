Models

  • Customers: Represents customers in the system
    • full_name: CharField, maximum length 100
    • age: IntegerField
    • email: EmailField
  • Doctors: Represents doctors with their details
    • name: CharField, maximum length 100
    • specialization: CharField, maximum length 100
    • available: BooleanField, default=True
  • Category: Represents service categories
    • name: CharField, maximum length 50
    • description: TextField
  • Service: Represents services offered
    • name: CharField, maximum length 100
    • category: ForeignKey to Category
    • price: DecimalField
  • Orders: Represents customer orders for services
    • customer: ForeignKey to Customers
    • service: ForeignKey to Service
    • order_date: DateTimeField
  • Money: Represents financial transactions
    • amount: DecimalField
    • transaction_date: DateTimeField
  • MainUsers: Represents main system users
    • username: CharField, maximum length 50
    • password: CharField
  • CustomGroup: Represents custom user groups with permissions
    • group_name: CharField, maximum length 50
    • permissions: ManyToManyField to Permissions
